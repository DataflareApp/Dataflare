# Dataflare

Dataflare is a simple, easy-to-use database manager. It is easy to connect to your databases, manage Table, view Data, write SQL and run Query.

![Dataflare](https://github.com/user-attachments/assets/78385d49-d686-4e21-87a2-2998c9f58cea)

## Features

- **Manage Connections** — Save multiple database connections and easily switch between databases.
- **View & Edit Data** — Easily filter, sort, insert, update, and delete data.
- **SQL Editor** — Built-in editor with real-time saving, syntax highlighting, and auto-complete.
- **Schema Manager** — View the structure and relationships of all tables in one place.
- **Dashboard** — View or analyze your data through beautiful charts.
- **Data Generation** — Quickly insert large amounts of random data for testing.
- **SQL Preview** — Review the SQL you are about to run to prevent unexpected behavior.
- **Read-only Mode** — View your data more securely and prevent accidental changes.
- **AI Enhanced** — Generate SQL queries, explain complex code, and get intelligent suggestions powered by AI.
- **Quick Search** — Quickly navigate to your Database, Table, or Query.

## Install

#### macOS

```bash
brew install --cask dataflare
```

#### Windows

```bash
winget install Dataflare.Dataflare
```

#### Linux & other platforms

Please visit Dataflare website to download: https://dataflare.app/download
